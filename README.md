# 📊 Dashboard de Vendas Profissional

Aplicação web desenvolvida com **Python** e **Streamlit** para análise de vendas, monitoramento de indicadores e visualização interativa de dados.

---

## 🚀 Status do Projeto

🚀 Projeto publicado via Streamlit Community Cloud

🔗 Acesse: [https://dashboard-vendas-pro-zaj5qlcnq25abddvpqkcxz.streamlit.app/]

---

## 📌 Funcionalidades

* 📈 Visualização de KPIs de vendas
* 💰 Análise de faturamento
* 🧮 Cálculo de ticket médio
* 📊 Gráficos interativos com Plotly
* 🔎 Filtros por período, produto e região
* 📅 Análise temporal de desempenho
* 🧩 Arquitetura modular (código organizado em camadas)

---

## 🛠️ Tecnologias Utilizadas

* Python
* Streamlit
* Pandas
* Plotly
* SQLite
* Pytest

## ▶️ Como Executar Localmente

### 1. Clone o repositório

```
git clone https://github.com/moisesneivert/Dashboard-Vendas-Pro.git
```

### 2. Acesse a pasta

```
cd Dashboard-Vendas-Pro
```

### 3. Crie um ambiente virtual

```
python -m venv .venv
```

### 4. Ative o ambiente

**Mac/Linux**

```
source .venv/bin/activate
```

**Windows**

```
.venv\Scripts\activate
```

### 5. Instale as dependências

```
pip install -r requirements.txt
```

### 6. Execute a aplicação

```
streamlit run app.py
```
